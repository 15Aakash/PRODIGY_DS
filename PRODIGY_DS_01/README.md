# PRODIGY_DS_01 – Population Data Visualization

## Problem Statement
Analyze and visualize population data to understand the distribution of population across different categories.

## Objective
The goal of this task is to explore population-related data and present insights using clear and meaningful visualizations.

## Key Steps
- Data loading and inspection
- Data cleaning and preprocessing
- Visualization using bar charts
- Interpretation of population distribution

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Files
- `PRODIGY_DS_01.ipynb` – Jupyter Notebook containing population data analysis and visualization
