# PRODIGY_DS_05 – Traffic Accident Data Analysis

## Problem Statement
Analyze traffic accident data to identify key factors contributing to road accidents.

## Objective
The objective of this task is to explore accident patterns based on road conditions, weather, and other influencing factors.

## Key Steps
- Data exploration and cleaning
- Visualization of accident frequency
- Analysis of road and environmental conditions
- Insight generation

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Files
- `PRODIGY_DS_05.ipynb` – Jupyter Notebook containing traffic accident analysis
